Sending and Receiving Messages in |SQS|

After you create a queue in |SQS|, you can send a message to it and then consume it. To learn more, see Tutorial: Sending a Message to an Amazon SQS Queue <sqs-send-message> and Tutorial: Receiving and Deleting a Message from an Amazon SQS Queue <sqs-receive-delete-message>.

In this example, you use the |sdk-ruby| with |SQS| to:

  1. Send a message to a queue by using send_message.

note

If your queue is a FIFO queue, you must include a message_group_id parameter in addition to the id and message_body parameters.

  1. Receive the message in the queue by using receive_message.
  2. Display information about the message.
  3. Delete the message from the queue by using delete_message.

Prerequisites

Before running the example code, you need to install and configure the |sdk-ruby|, as described in:

  • aws-ruby-sdk-setup-install
  • aws-ruby-sdk-setup-config

You also need to create the queue my-queue, which you can do in the |SQS| console.
